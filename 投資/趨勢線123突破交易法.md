# 錯誤的突破
錯誤的突破原因如下：
1. 圖沒畫好
2. 有更大的整理結構
	![[Pasted image 20220714175204.png]]	
3. 技術不好？

# 正確的突破
1. 突破壓力線
2. 回踩壓力線
3. 突破前高
![[Pasted image 20220714175513.png]]

# 看懂正確的突破了，該如何操作？
123 三點均可操作，不過風險不同。
1. 突破交易，風險大
2. 趨勢交易，不一定會回踩，但獲利空間大
3. 趨勢確立交易，勝率最高
- 止損位置：P點 
	-> 因為要有Higher Low 才是上漲趨勢，若跌破前低則失效。
